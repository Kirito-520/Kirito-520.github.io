# Python 知识点总结

2020-08-17

1.列表[ ]

  • 列表名[ 索引名] 索引列表的元素
    例如：d[0]
  • 用列表名p[起：止]表示切片，从列表里切出相应的元素，前闭后开
    c = [1,2,3,4,5,6,7]
    如：c[0:2]切出[1,2]
  • 用列表名【起：止：步长】 带步长的切片，步长有方向
    如：切出【5，4，3，2】用c【4：0：-1】 。切出【5，4，3，2，1】用c【4：：-1】
  • 修改：列表名【索引号】 = 新值
  • 删除：del 列表名【索引号】
  • 插入：列表名.insert(插入位置索引号，新元素)
    2.元祖
    元祖() 一旦定义就不能改变，如：f=(1,2,3)
    3.字典{ }
  • 字典里存放着{键：值，键：值，键：值} n个键值对
    如："""dic={1:"123",“name":zhangsan,"height":180}"""
