# Python 知识点总结

2020-08-17

1.列表[ ]

  • 列表名[ 索引名] 索引列表的元素
    例如：d[0]
  • 用列表名p[起：止]表示切片，从列表里切出相应的元素，前闭后开
    c = [1,2,3,4,5,6,7]
    如：c[0:2]切出[1,2]
  • 用列表名【起：止：步长】 带步长的切片，步长有方向
    如：切出【5，4，3，2】用c【4：0：-1】 。切出【5，4，3，2，1】用c【4：：-1】
  • 修改：列表名【索引号】 = 新值
  • 删除：del 列表名【索引号】
  • 插入：列表名.insert(插入位置索引号，新元素)
    2.元祖
    元祖() 一旦定义就不能改变，如：f=(1,2,3)
    3.字典{ }
  • 字典里存放着{键：值，键：值，键：值} n个键值对
    如：dic={1:"123",“name":zhangsan,"height":180}
  • 用字典名【键】索引字典中的值
    dic【“name”】表示字典dic中键“name”对应的值为zhangsan
  • 修改：字典名【键】= 新值
  • 删除： del 字典名【键】
  • 插入：字典名【新键】=新值

4.条件语句

  • if 条件成立：
    执行任务
  • if 条件1成立：
    else:
    执行任务2
  • if 条件1成立：
    执行任务1
    elif 条件2成立：
    执行任务2
    elif 条件n成立：
    执行任务n
    else：
    执行任务n+1
  1. 循环语句
    for 变量 in range(开始值，结束值)：
    执行某些任务
    for 变量 in 列表名:
    执行某些任务
    while 条件：
    执行某些任务
    终止循环用break

turtle模块
import turtle # 导入turtle模块
t = turtle.Pen() # 用turtle模块的Pen类，实例化一个叫做t的对象
t.forward(像素点) # 让t 向前走多少个像素点
t.backward（像素点）# 让t向后走多少个像素点
t.left(角度) # 让t左转多少个角度
t.right(角度) #让t右转多少角度
t.reset() #让t复位

6.函数与模块
函数：执行某些操作的一段代码
定义函数： def 函数名（参数表）：
函数体
使用函数： 函数名（参数表）

  • 模块 ：函数的集合，先导入，再使用，用模块.函数名调用
  • 包：包含多个模块

7.类 对象 面向对象
类：物以类聚人以群分，是函数的集合，可实例化对象的模具
实例化：对象=类() 如：t = turtle.Pen()
对象：是类实例化出的实体，对象实实在在存在，完成具体工作。
面向对象：程序员反复修改优化类，类实例化对象，对象调用类里的函数执行具体操作。
类的定义：首字母要大写
class 类名（父亲名）：
pass
对象调用类里的函数，用对象.函数名
对象调用类里的变量，用对象.变量名
类内定义函数时，如调用自身或父类的函数与变量，必须用self.引导 ，应写为 self.函数名或者self.变量名
8. 文件写操作 import pickle
开：文件变量 = open(" 文件路径文件名”，"wb")
存：pickle.dump(待写入的变量，文件变量)
关：文件变量.close()
文件读操作 import pickle
开：文件变量 = open(" 文件路径文件名”，"rb")
存：放内容的变量=pickle.load(文件变量)
关：文件变量.close(()
